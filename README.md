# Son Nguyen's Personal Website
This website represent my professional online presense. It introduces who I am, what skills and education I had, and a list of my 3 most recent logo works in SIAT to show my skills. It is also used to show the visualization and creative art that I made using SVG.

## Technologies Used
- **HTML**: Include content and structure of the site.
- **CSS**: For styling elements on the website, including media queries for responsiveness.
- **JavaScript**: Including function that helps draws the bars in my visualization.
- **SVG**: For creating and displaying the visualization and creative art.

## Site Navigation
The website consists of two pages that can be accessed via the navigation bar on the top of the website.

### Home Page
- **Introduction**: Brief introduction paragraph, education background, and skills.
- **Selected Works**: My 3 most recent logos that I made in SIAT.

### Visualizations Page
- **SVG Visualization**: Bar chart showing how many hour of sleep I got per day last week
- **Creative SVG Art**: An artwork that I made using SVG
